PRINT

Druckt Textdateien während der Verwendung anderer IBM DOS-Befehle.

PRINT [/D:Gerät] [/B:Größe] [/U:Takte1] [/M:Takte2] [/S:Takte3]

[/Q:WGröße] [/T] [[Laufwerk:][Pfad]Dateiname[ ...]] [/C] [/P]

/D:Gerät Gerät, auf dem gedruckt werden soll.

/B:Größe Größe des internen Buffers (in Byte).

/U:Takte1 Maximale Anzahl von Takten, die abgewartet wird, bis ein Drucker

verfügbar wird.

/M:Takte2 Maximale Anzahl von Takten, die abgewartet wird, bis ein Zeichen

gedruckt wird.

/S:Takte3 Zeitspanne (in Takten), die IBM DOS für das Drucken im

Hintergrund zugeordnet wird.

/Q:WGröße Maximale Anzahl von Dateien in der Druckwarteschlange.

/T Entfernt alle Dateien aus der Druckwarteschlange.

/C Bricht Drucken des vorangestellten und aller folgenden

Dateinamen ab.

/P Fügt den vorangestellten und alle folgenden Dateinamen der

Druckwarteschlange hinzu.

PRINT ohne Parameter zeigt den Inhalt der Druckwarteschlange an.
