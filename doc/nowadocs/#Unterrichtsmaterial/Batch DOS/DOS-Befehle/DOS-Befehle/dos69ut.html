MOVE

Zum Versetzen einer Datei/mehrerer Dateien folgendes eingeben:

MOVE [/Y | /-Y] [laufwerk:][pfad]dateiname1[,...] ziel

Zum Umbenennen eines Verzeichnisses folgendes eingeben:

MOVE [laufwerk:][pfad]verzname1 verzname2

/Y Bestätigung bei Überschreiben des Ziels wird unterdrückt.

/-Y Bestätigung bei Überschreiben des Ziels ist erforderlich.

Der Schalter /Y kann in der Umgebungsvariablen COPYCMD definiert sein.

Diese Einstellung kann durch Eingabe von /-Y außer Kraft gesetzt werden.

[laufwerk:][pfad]dateiname1 Gibt die Position und den Namen der Datei bzw.

der Dateien an, die versetzt werden soll(en).

ziel Gibt die neue Position der Datei an. Die Zielangabe

kann aus Laufwerkbuchstabe und Doppelpunkt, Ver-

zeichnis oder aus einer Kombination dieser Angaben

bestehen. Wird nur eine Datei versetzt, kann auch

zum gleichzeitigen Umbenennen ein neuer Dateiname

angegeben werden.

[laufwerk:][pfad]verzname1 Name des umzubenennenden Verzeichnisses.

verzname2 Neuer Name des Verzeichnisses.
