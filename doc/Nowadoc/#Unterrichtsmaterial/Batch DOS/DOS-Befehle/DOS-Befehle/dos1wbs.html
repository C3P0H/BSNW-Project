PATH

Legt den Suchpfad für ausführbare Dateien fest oder zeigt diesen an.

PATH [[Laufwerk:]Pfad[;...]]

PATH ;

Geben Sie 'PATH ;' ein, um die Suchpfadeinstellung zu löschen und um IBM DOS

nur im aktuellen Verzeichnis nach ausführbaren Dateien suchen zu lassen.

Der Befehl PATH ohne Parameter zeigt den aktuellen Suchpfad an.
