COPY

Kopiert eine oder mehrere Dateien an eine andere Position.

COPY [/A | /B] Quelle [/A | /B] [+ Quelle [/A | /B] [+ ...]] [Ziel

[/A | /B]] [/V] [/Y | /-Y]

Quelle Bezeichnet die zu kopierende(n) Datei(en).

/A Weist auf eine ASCII-Textdatei hin.

/B Weist auf eine Binärdatei hin.

Ziel Bezeichnet Verzeichnis und/oder Dateiname der neuen Datei(en).

/V Überprüft, daß die neuen Dateien richtig aufgezeichnet wurden.

/Y Unterdrückt die Bestätigungsaufforderung vor dem Überschreiben

einer vorhandenen Zieldatei.

/-Y Zeigt eine Bestätigungsaufforderung an, bevor eine vorhandene

Zieldatei überschrieben wird.

Der Schalter /Y kann in der Umgebungsvariablen COPYCMD voreingestellt werden.

Um Dateien aneinanderzuhängen, eine Zieldatei, aber mehrere Quellendateien

angeben (mit Platzhalterzeichen oder in der Form Datei1+Datei2+Datei3).
