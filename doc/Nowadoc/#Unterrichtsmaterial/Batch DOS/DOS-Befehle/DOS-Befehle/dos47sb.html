INTERLNK

Status der durch INTERLNK/INTERSVR umgeleiteten Laufwerke wird angezeigt.

INTERLNK [Klient=[Server]]

Klient Gibt das Klientenlaufwerk an, das zu einem Server-Laufwerk

umgeleitet werden soll. Umleitung wird aufgehoben, wenn

kein Server-Laufwerk angegeben wird.

Server Gibt ein Server-Laufwerk an, das zu einem

Klientenlaufwerk umgeleitet wird.

Durch die Eingabe von INTERLNK ohne Parameter wird der Interlnk-Status

angezeigt.
